# Webchat Legal Triage - Stakeholder Register
| Stake Holder | Interest & Influence | Expectations | 
| ------ | ------ | ------ |
| Client | High interest, low influence | To be able to start new matters, view open and closed matters. Access and change their details. Be able to have live chat conversations with legal advisors |
| Legal Advisor | High interest, medium influence | To be able to view assigned matters and messsages (filtering by read and unread). Be able to edit client and matter details |
| Supervisor | Medium interest, high influence | To be able to view dashboard and statistics. Be able to assign matters to legal advisors. Be able to audit any and all matters. |
| Administrator | Low interest, high influence | Full access control admin access |
